Learning to use fonts in CSS

CSS Cheat Sheet For Fonts

Fonts are a crucial part of learning CSS and how to style your own websites! There is a lot to go over and point of this helpful table is to help those new to CSS to help understand fonts better and how to make your own website look stylish and cool!

This is all for you to know:

Selector Description Output
font-family Specifies a specific list of fonts of either 1 or 2 for the particular element. Remember the first font selected will take predence over others. font-family: 'Merriweather', 'Arial', serif; (Father vs Father )
font-weight Changes the appearance of the font, whether it is lighter or heavier in thickness. font-weight: #px; (font-weight of 200: Word )
font-style Changes the style of a font to either italics or obliqued (meaning font is slightly angled, must put in degree) font-style: italics/oblique #deg; ( Obliqued 10 degrees )
letter-spacing/word-spacing Adds spacing between either letters of a word or between words within in a paragraph/sentence. letter/word-spacing: #em ( This is a sentence with word spacing )
